
# Getting started
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
Project using TypeORM and GraphQL

## Installation

```bash
$ npm install
```

## Running migration
```bash
$ npm run typeorm migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Open in browser
[http://localhost:3333/graphql](http://localhost:3333/graphql)
